Sampletexter:
  A Python programm that converts all words of a .docx document in "sampletext".
Exceldefaulter:
  A Python programm that makes an excel list of names(using names module) with some random values, 
  a random generate date that takes into account Leap years and a random generated City. 
  Also uses City Scrapper in order to obtain a bunch of cities.
WordExcelOPT:
  A Python programm that takes a list of names with several values (from excel defaulter), 
  feeds that data into a List of objects (called Persona)
  and then uses this values to generate multiple ( as many as people are on the list).docx 
  where it replaces several pieces with information from the excel. Pandas and docx are used
