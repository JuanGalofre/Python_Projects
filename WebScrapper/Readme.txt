DictScrapper:
  It is a webscrapper application for academic purposes. Due to long connection times and search engine mistakes, is way easier to scrap the definitions out of the 
  webapge. It uses beautiful soup 4 and requests in order to search for a word in a German-Spanish mode. It takes the HTML elements where the locations and type of word
  are located and displays it to an user. Can be modfied to accept lists of words(V2)
CityScrapper:
  A Python code which takes the cities from a city list that can be found in Wikipedia, and creates an Excel with it. Beautiful Soup, pandas, xlswritter and requests are used
